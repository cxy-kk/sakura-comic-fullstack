sakura-comic-vue
简介
动漫网站前端部分
一个用于追番的动漫网站，实现了动漫视频的更新与展示，用户注册和登录，评论和回复，视频收藏的功能，添加了直播功能
技术栈，前端： vue3 + vite + element plus, 后端：flask + gunicorn + mysql
运行
安装运行所需的包：npm install

测试运行： npm run dev
